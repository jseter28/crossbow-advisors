### What works best
- **Orientation:** landscape / wide. The band is full-bleed and uses
  `background-size: cover`, so the image is cropped to fill — give it room.
- **Resolution:** ≥ 2000px wide (2400–2560px ideal) so it stays crisp on large displays.
- **Mood:** dark, moody, dusk/night financial-district skylines read best — the
  band lays a charcoal gradient over the photo so light text stays legible.
  Bright midday shots will need a heavier overlay (easy to tune).
- **Format:** `.jpg` (smaller). If you only have `.png`/`.webp`, tell me and
  I'll point the CSS at that extension.

Until real photos are added, the band shows `skyline-placeholder.svg`
(an on-brand silhouette) so the effect is visible.
